# NCUtils #

A small collection of everyday Java utilities


Using with Gradle:

    repositories {
        maven {
            url 'https://dl.bintray.com/nelsoncrosby/maven'
        }
    }

    dependencies {
        compile 'io.github.nelsoncrosby:NCUtils:0.1.1'
    }
